# ShikshaAI Project Blueprint

## 1. Vision & Core Logic
- Left Panel: "Tina" AI Tutor Avatar Studio. Handles multilingual input and audio responses (English, Hindi, Telugu). 
Includes smooth idle, thinking, and speaking visual animations.
- Right Panel: Dynamic Visual Knowledge Panel. Renders structured markdown explanations, key points, related Pexels images,
 facts, and comparison tables strictly in English to build technical comprehension.
